
## Run Locally

Clone the project

```bash
  git clone https://github.com/AmanTrivedi1/customlink.git
```

Go to the project directory

```bash
  cd customlink
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
